Ajuda e Atalhos

Como usar o mapa

  • 🔍 Buscar: Digite o nome de uma congregação para filtrar marcadores
  • 🎯 Encaixar: Clique no botão 🎯 para visualizar todos os marcadores
  • 📍 Marcadores: Clique em qualquer congregação para ver detalhes
  • 🗺️ Basemaps: Use os botões no canto inferior direito para trocar o mapa
  • 📋 Copiar: Use o botão no popup para copiar as coordenadas

Navegação GPS

Clique em um marcador e use os botões para abrir em:

  • 📍 Waze - Navegação GPS em tempo real
  • 🗺️ Google Maps - Mapa e instruções de direção

Atalhos de Teclado

  • ESC: Limpar busca
  • F: Encaixar todos os marcadores
